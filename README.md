## Run Locally

Clone the project

```bash
  git clone https://github.com/abhaypsamrat/Task-Manager-NodeJS.git
```

Go to the project directory

```bash
  cd TASK-MANAGER
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  nodemon app.js
```
